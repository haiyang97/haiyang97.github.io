解释/用法：1——6级标题

解释/用法：斜体

解释/用法：加粗

<del></del>：加删除线标签

解释/用法：给文字加删除线

<sup></sup>：上标标签

解释/用法：把文字上标

<u></u>：下划线标签

解释/用法：给文字加下划线

<center></center>：居中标签

解释/用法：把文字居中

<br>：换行标签

CSS引入及优先级

CSS选择器的优先级/权重值

优先级分权重值而定，而对于新手来说，计算权重值过于复杂，可先理解为：选择器选择的范围越小越精确，优先级就越高。

  • 权重值：

    -通用选择器（*） 权重值：0

    -标签选择器（div p…） 权重值：1

    -类选择器（.act .nav…）权重值：10

    -ID选择器（#btn #box…）权重值：100

    -行内样式 权重值：1000

如上图，权重值相加，权重值高的优先级高。

CSS文本属性

  • 文字颜色：color:red;
  • 字体类型：font-family:"微软雅黑";
  • 字体大小：font-size:24px;
  • 文字加粗：font-weight:bold;
  • 文字倾斜：font-style:italic;
  • 首行缩进：text-indent:60px;
  • 水平对齐方式：text-align:center;
  • 行高：line-height:100px;
  • 垂直居中：height:100px;line-height:100px;（容器的高设置为100px，再把文字的行高也设置为100px，文字就会垂直居中了）
  • 文本修饰：text-decoration;（给文字加了下划线）如果给文本加了超链接，默认会有下划线，可添加text-decoration:none;来取消下划线

补充：

  • 排列问题：

    浮动元素在排列时，只会参考前一个元素的位置。

  • 左右顺序问题：假如有三个左浮动的元素，如果把他们改为右浮动，责顺序会被反转。

  • 重叠问题：

    1.浮动元素不会覆盖文字内容

    2.浮动元素不会覆盖图片内容（因为图片本身也属于文本，可以把图片看作一个特殊的文本）

    3.浮动元素不会覆盖表单元素（输入框、单选按钮、复选框、按钮、下拉选择框等）
